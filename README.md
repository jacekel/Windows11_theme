#EpicLice is a warm, energizing Windows 11 by Microsoft.com | theme by Jacek Łaszewski and Copilot. It pairs a sunlit color palette with custom wallpapers, icon swatches, and a ready-to-use `.theme` file so you can quickly apply a cohesive look across your desktop. 

## Contents

- `theme/EpicLice.theme` — main theme file ready to install  
- `oeminfo/oeminfo.ini` — optional OEM info sample  
- `wallpapers/` — high-resolution images (1920×1080, 2560×1440, 3840×2160)  
- `icons/` — SVG and PNG icon set and color swatches (8×8, 16×16, 24×24, 32×32)  
- `install/` — optional install helpers (`.bat`)  
- `LICENSE` — MIT license text  
- `CHANGELOG.md` — release notes and history
